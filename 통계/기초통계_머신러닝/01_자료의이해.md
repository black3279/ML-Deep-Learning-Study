# 통계및머신러닝
## 1) 자료의 이해
### 모집단과 표본
- 모집단 : 통계분석 방법을 적용할 관심 대상의 전체 집합 => 모수
- 표본 : 직접적인 조사 대상이 된 모집단의 일부 => 통계량
- 개체(Item) : 연구자 또는 관찰자가 관심을 갖는 대상
- 요인(Factor) : 개체에 관한 특성 중 연구자가 특별히 관심을 갖는 특성
- 변수 : 요인을 구성하고 있는 요소
- 각 칼럼을 변수라 하며 변수들에 대한 측정값을 관측치라 한다
* 머신러닝에서는 각각의 열 데이터를 특성(feature), 속성(attribute) 라 한다
* 머신러닝에서는 각각의 행 데이터를 instance 라 한다

### 범주형 : 특성으로 문자열이 들어가는 데이터 종류
### 수치형 : 특성으로 숫자가 들어가는 데이터 종류
  - 머신러닝에서 맞추고 싶은 값을 Target value, class, label 이라 말한다
  - 통계에서는 종속변수라 한다
- 범주형데이터 처리 시에는 빈도를 기반으로 처리한다
- 독립변수 (설명변수, 예측변수, 위험인자, 공변량 - 연속형, 요인 - 범주형)
- 종속변수 (반응변수, 결과변수, 표적변수)
- 종속변수가 범주형일 때 이를 분류모델이라 하며 수치형일 때는 회귀분석이라 한다

### 수치형 변수의 종류
- 이산변수 : 하나하나 셀수있는 정수 값 (유한한 범위)
- 연속변수 : 측정치 사이에 갭이 없는 연속변수 (무한한 범위)

### 변수의 타입
- 범주형 : 정성적, 질적 자료
  - 명목형 : 단순히 범주를 표시
  - 순서형 : 범주의 순서를 상대적으로 비교가능
- 수치형 : 정량적, 양적 자료
  - 이산형 : 셀수 있는 형태의 자료
  - 연속형
    - 등간형(interval) : 비교할 수 있도록 단위가 정해졌으며 0은 하나의 값을 의미한다
    - 비율형(ratio) : 0은 없음을 의미하는 경우이며 사칙연산이 가능하다

- 상호배타성, 서열비교, 표준측정단위 덧셈뺄셈 여부, 자연적 0의 존재에 따라 하기와 같이 나눈다
  - 명목척도 : 상호배타성만 만족
  - 서열척도 : 상호배타성, 서열비교 만족
  - 등간척도 : 상호배타성, 서열비교, 표준측정단위 덧셈뺄셈 만족
  - 비율척도 : 상호배타성, 서열비교, 표준측정단위 덧셈뺄셈, 자연적 0의 존재를 만족

### 변수의 활용
- 수치형 데이터의 경우 분포 분석으로 설명하며 주요 항목은 범위, 평균, 분산, 표준편차 등이고 특정 분포를 대부분 가정한다
- 범주형 데이터의 경우 범주별 출현빈도에 기반해 분석하며 주요 항목은 빈도, 비율, 누적비율 등이고 분포 가정 없이 빈도에 기반한 확률을 사용한다
- 독립변수 x, y 순으로 하기와 같이 분석 기법을 적용한다
- x, y 모두 수치형이면 상관분석, 회귀분석을 한다
- x 가 범주형이며 이를 통해 y 수치형을 추정하면 t-test, ANOVA 를 적용한다
- x 는 수치형이며 이를 통해 y 범주형을 추정하면 로지스틱 회귀분석을 적용한다 (분류모델)
- x, y 모두 범주형이면 카이제곱 검정을 적용한다

### 자료의 정리 방법
- 통계표 : 자료의 전체적인 특성을 파악하기 위하여 원시 데이터의 특성을 정리 요약하는데 사용하는 도구이다
- 보고서 내용은 하기로 구성한다
  - 서론 : 분석의 타당성과 방법을 설명한다
  - 본론 : 데이터 요약 통계표, 분석내용, 결과값이 나온다
  - 결론 : 분석결과, 의사결정 내용, 시사점이 나온다

- 도수분포표 : 자료를 변수의 계급 또는 범주로 나누고 각 계급에 속하는 측정치의 도수를 나타내는 통계표이다
- 상대도수 : 각 계급의 도수가 총 도수에서 차지하는 비율이다
- 히스토그램 : 표로 된 도수 분포를 정보 그림으로 나타낸 것이다, 가로축이 계급이며 세로축이 도수를 뜻한다
- 가로, 세로 빈도를 가지고 분할표로 표현한 것을 분할표, 교차표라 하며 카이제곱 검정에 쓰인다
- 두 개 이상의 변수를 동시에 고려하며 관측개체의 빈도를 정리한 것이라 할 수 있다

- 질적자료의 해석은 도수분포표, 상대도수 막대그래프, 원형그래프 로 할 수 있다
- 양적자료의 해석은 줄기 잎 그림, 도수분포표, 히스토그램으로 할 수 있다

### 차트
- 범주형 데이터를 표현하는 방법에는 파이그래프와 띠그래프가 있으며 일변량 차트이다
- 수치형 데이터를 표현하는 방법에는 히스토그램과 한개인 Box plot 이 있으며 일변량 차트이다
- 히스토그램은 중심확인가능하며 대칭여부, 그룹존재여부, 이상치존재여부를 알 수 있다
- Boxplot 은 단일 값 유무, 흩어진 정도, 대칭여부, 이상치 존재유무, 그룹별 평균과 흩어진 정도를 알 수 있다
- 산점도는 x, y 둘 다 수치형이 오며 관계성, 그룹존재 여부, 이상치 존재 유무를 알 수 있으며 인과관계는 알 수 없다
- 범주형 & 수치형을 나타내는 방법에는 Boxplot 과 Bar Chart 가 있으며 다변량 차트이다
- y 축이 이름이며 x 축은 수치형인 경우거나 Boxplot 이 두개 이상인 경우이다
- 수치형과 수치형을 나타내는 방법에는 산점도가 있으며 다변량 차트이다
- 범주형 & 범주형을 나타내는 방법에는 Mosaic plot 이 있다 (빈도를 counting 하여 색깔로 표현)
- 범주형 & 범주형 & 수치형을 나타내는 방법에는 Stacked Column Chart 가 있다 (범주를 색상으로, 각 선호도 비율을 수치로 표현한다)
