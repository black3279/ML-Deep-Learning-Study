# 통계및머신러닝
## 2) 기초통계분석
- 위치통계량 (중심 경향성) : 대표값과 그 위치
- 변이통계량 : 흩어진 정도
- 모양통계량 : 분포 모양

- 평균은 유일한 값을 가지며 극단적인 값이 민감하다
- 기하평균은 직접 값을 기준으로 성장율 등을 표현할 때 사용하며 n 개 양수 곱의 n 제곱근 이다
- 조화평균은 평균적인 변화율이며 주어진 수들의 역수의 산술평균에 다시 역수를 취한 것이다
- 가중평균은 차지하는 비율이 다른 경우에 n 값을 곱하여 사용한다

- 최빈값은 자료에 따라 존재하지 않을 수 있고 유일한 값이 아닐 수 있다
- 명목자료와 서열자료의 경우 평균과 중앙값을 계싼할 수 없으므로 최빈값을 대표자료로 사용한다
- 기대값은 모르는 데이터에 대한 평균 값이라 할 수 있다
- 변이 통계량에는 범위, 중간범위, 평균절대편차가 있다
- 자유도(df) 란 편차가 0 이 되어야 하므로 특정된 데이터들을 제외한 자유로운 데이터의 개수를 활용하는 것이다
- 표본은 모집단의 subset 이므로 모수의 분산을 구하기 위해서는 과소추정을 해결하고자 n-1 (자유도) 를 활용한다
- 자료가 하나인 경우 자료 하나로부터 퍼진 정도를 알 수 없다
- 표준편차를 평균으로 나눈 값을 변동계수(변이계수)라 한다
- 변동계수를 coefficient of variation(CV), 혹은 상대표준편차 (relative standard deviation) 이라고 한다
- scale 이 다를 때 서로 다른 데이터 간 편차를 비교하는 방법으로 변동계수가 사용된다
- 상자 수염 그림이란 최소값, 1사분위수, 2사분위수, 3사분위수, 최대값을 표현한 그림이다
- 양의 왜도는 오른쪽으로 꼬리가 긴 분포며 음의 왜도는 왼쪽으로 꼬리가 긴 분포이다
- 첨도는 0 보다 크면 정규분포보다 뾰족하며 0 보다 작은 경우 정규분포보다 납작하다

## 3) 확률과 분포
- 결과를 알지 못하나 가능한 경우를 알고 있고 반복 가능한 경우 확률 실험 혹은 확률 시행이라 한다
- 표본 공간 sample space 를 S 또는 오메가 라고 한다
- 서로가 배반사상(두 사건이 동시에 발생하지 않는 사건)인 경우 합집합은 더한 것이며 교집합은 곱한 것과 같다
- P(A) = P(A|B) = P(A|B여집합) 일 때 두 사건 A,B 는 서로 독립이라 한다
- 독립사건 인 경우 교집합은 P(A)*P(B) 와 같다
- P(A|B) = P(A) or P(A교집합B) = P(A)*P(B) or P(B|A) = P(B) 중 하나를 만족하면 독립사건이라 할 수 있다
- 이벤트 앞단을 사전확률, 이벤트 발생 뒷단을 조건부 확률, 이 둘의 곱을 결합확률, 결합확률의 합으로 나타내는 주변확률이 있다
- 사후확률 = 사전*조건부 / 주변확률 을 전확률공식(베이즈정리)이라 한다
- 사건의 원인을 제공하는 확률을 사전확률, 발생 이후의 확률을 사후확률 이라 한다
- 확률변수는 확률분포를 따르며 확률분포함수에는 확률질량함수(PMF, 이산변수), 확률밀도함수(PDF, 연속변수)가 있다
- 확률밀도함수는 면적을 구하기 위한 것이며 무한한 n 을 갖으므로 대입하면 0으로 수렴한다
- 이산확률분포의 분산은 편차제곱의 기대값이라 할 수 있다
- 상수의 기대값은 상수이며 곱셉은 그대로 뺄수있다
- 상수의 분산은 0이며 곱셈은 제곱으로 나오고 서로 독립인 경우 합을 분리할 수 있다
- 결합확률분포는 x y 값의 모든 쌍에 대해 결합 확률을 나열한 표이다
- 확률분포의 독립성이란 어느 한 변수가 어떤 값을 취한 사실이 다른 변수에 전혀 영향을 주지 않는 것을 말한다
