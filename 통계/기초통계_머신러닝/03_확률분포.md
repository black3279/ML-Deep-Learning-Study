# 통계및머신러닝
## 3) 확률분포
- 균등분포 : 모든 곳에서 값이 일정한 분포
- 이항분포 : 결과가 2개로 나오며 n 번의 시행은 독립적이고 베르누이 실험 또는 베르누이 시행에 기초한다
  - 이항분포, 포아송분포, 초기하분포가 이항분포에 해당한다
- P(x) = nCx* p 의 x제곱 * (1-p) n-x제곱
- 성공확률이 0.5에 가깝거나 시행횟수 n 이 크면 이항분포는 좌우대칭을 이룬다
- p 가 1/2 작고 n 이 작은 경우 이항분포는 오른쪽 꼬리분포를 나타낸다
- 포아송분포 : 일정한 단위 시간에서 특정 사건이 발생한 횟수를 이용하는 경우 사용할 수 있는 이산형 확률분포이다
  - 구간마다 발생하는 사상은 독립적이어야 하며, 사상 발생확률은 구간의 길이에 비례하고 구간마다 확률분포는 일정하다
  - 평균 = 분산 = 사건발생(람다)
  - 람다가 커질수록 정규분포를 따르며 일반적으로 크지 않다면 오른쪽 꼬리 분포를 따른다
- 지수분포 : 사건이 서로 독립이며 포아송분포를 따른다면, 다음 사건까지 대기시간이 지수분포이다
  - 평균 = 1/람다 , 분산 = 1/람다제곱 인 연속확률변수로 시간이 지날수록 확률이 작아진다
  - 람다가 클수록 대기시간은 짧아진다
  - 무기억성 : 특정 사건이 x 시간 동안 발생하지 않았을 때 이후 t 시간 동안 발생 x 확률은 p(t) 와 같다
- 정규분포 : 연속확률변수를 기술하는 가장 중요한 확률분포로 종같이 보이는 부드러운 곡선 형태이다
  - 중심극한정리 : 하나의 모집단에서 여러개의 표본 추출하여 표본들의 평균의 평균을 구했다고하자,
  - 모집단에서 추출한 값이므로 모집단의 분포와 상관없이 해당 값들은 정규분포를 따르게 된다
  - 표본의 평균의 평균은 모집단의 평균의 불편추정량이 된다
  - 형태와 위치는 평균과 표준편차가 결정하며 곡선 밑의 면적은 1이고 오른쪽/왼쪽은 각각 0.5 이다
  - 평균 주위로 +-시그마 범위에 각각 68, 95, 99% 데이터가 있다 (1.96 -> 95%)
  - 표준정규분포 : 확률변수 z 가 평균 0 표준편차는 1인 정규분포이며 표준화를 통해 서로 다른 분포를 비교가능하다
  - Z = x - 모집단평균 / 시그마
- t 분포 : 좌우 대칭 종모양으로 중심은 0 이며 자유도에 따라 형태가 달라진다
  - 표본의 수가 적을 경우, 모집단의 분산을 모를 경우 샘플로 모집단의 평규능ㄹ 추론하기 위해 고안된 분포이다
  - 표준정규분포에 비해 첨도가 상대적으로 낮다
  - t = 표본의 평균 - 모집단의 평균 / 표준오차
- 카이제곱 분포 : 서로 다른 독립적인 표준정규분포의 제곱합은 자유도가 k 인 카이제곱 분포를 따른다
  - 모집단의 분산을 추론할 때 사용하며 심하게 왼쪽으로 쏠린 분포로 자유도에 따라 모양이 변하며 자유도가 커질수록 정규분포에 가까워진다
- F 분포 : 두 확률변수가 자유도가 각각 k1, k2이고 서로 독립인 카이제곱 분포를 따른다고 할 때 확률변수 F 는 자유도가 k1, k2 인 F 분포를 따른다
  - 두 모집단의 분산 차이를 비교할 때 사용한다
  - 이분산비를 활용하여 동질성 여부를 검정하거나 두 개 이상의 평균치 간의 차이 유무를 검정한다
  - 항상 양의 값만 갖는 왼쪽으로 치우친 비대칭 형태로 자유도가 커질수록 정규분포에 가까워진다
