# 통계및머신러닝
## 4) 추론 통계
- 점추정 : 모집단 특성을 단일한 값으로 추정하는 방법으로 적률방법과 최대우도 추정방법이 있다
  1) 일치추정량 : 샘플의 크기가 클수록 추정량이 모수에 근접하는 성질
  2) 불편추정량 : 차이가 없는 값
  3) 유효 효율 추정량 : 분산이 작을수록 점추정에 유리하다
  4) 충분추정량 : 대표하는 표본으로 부터 구한 통계량
- 구간추정 : 모수가 있을 것으로 예상되는 구간과 그 구간에 실제 모수가 있을 예상 확률을 구한다
- 대수의 법칙 : 표본 수가 많을수록 실사건의 확률이 통계적 예측에서 오차가 줄어간다
- 신뢰구간 : 모수가 특정확률로 포함될 것이라고 주장하는 범위
- 신뢰수준 : 모수의 참값이 두 신뢰한계 안에 포함될 것이라고 주장할 때 사용하는 확률
- 오차율 : 신뢰구간이 모평균을 포함하지 않을 확률 a 를 의미
  표본평균 +- t or z value * 표준오차 (= 표본의 표준편차 / n제곱근)
### 신뢰구간의 계산
- 모분산을 알 때 혹은 n 이 30이상일 때 : z 분포 사용 가능 => 표본 평균 - Z(a/2) * 시그마/n제곱근
- 모분산을 모르고 n 이 30 보다 작을 때 : t 분포 사용 가능 => 표본평균 - t(a/2) * 표본표준편차/n제곱근
- 표준오차란 표본 평균들의 표준편차로 표본 평균들의 퍼짐 정도를 나타낸다
- 표준오차 = 원시 데이터의 표준편차 / 평균값 계산에 사용한 데이터 수 제곱근
- 귀무가설이란 모집단에 대해 옳다고 주장하는 잠정적인 제안 또는 명제로 보수적인 주장, 차이가없다, 관계가없다와 같은 가설이다
- 대립가설이란 귀무가설의 주장이 틀렸다고 입증하려는 주장, 가설을 말한다
- 모수의 주장에 대한 = 부분은 항상 귀무가설에 포함시킨다
- 유의수준 a : 귀무가설이 참인데 대립가설을 선택하는 오류를 범할 최대 허용 한계를 말한다
- 유의수준을 넘는 범위를 기각역이라 부른다
- 제 1종 오류란 옳은 것을 옳지 않다고 판정할 확률이며 제 2종 오류는 옳지 않은 것을 옳다고 판정할 확률이다
- 유의수준은 더 위험한 오류인 1종 오류의 한계 허용치를 의미한다
- 검정력
  - 대립가설이 사실일 때 이를 사실로 결정할 확률
  - 검정력이 크면 2종 오류를 범할 확률이 낮아진다
- 유의확률 p-value : 귀무가설이 참일 때 검정통계량보다 극한값을 얻은 확률
- p-value 가 유의수준보다 작다면 귀무가설을 기각한다
- p-value 가 유의수준보다 크다면 귀무가설을 기각하지 못한다

### t-test
- 평균 비교를 통하 두 집단 사이의 차이를 검정한다
- 두 집단 모두 정규분포를 하고 있다는 가정하에 분석을 수행한다
- 단일표본, 대응표본, 독립 이표본 t검정 3가지 case 가 있다
- 대응표본 (paired t-test) 는 연관된 두 집단의 평균 차이값에 대한 단일 표본의 평균을 검정하기 위한 방법이다
- 샘플이 2개이며 독립적이지 않고 연관성이 있는 경우에 사용 가능하다
- 두 샘플은 샘플의 개수, 두 개 샘플의 순서가 같아야한다
- 독립 이표본 (two sample t-test) 은 서로 다른 두개 그룹 간 평균 검정을 하기 위한 방법이다
  - 독립성, 정규성, 등분산성을 가정한다
