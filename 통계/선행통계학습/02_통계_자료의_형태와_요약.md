# 러닝패킷: 통계의 기초
## 2) 자료의 형태와 요약2
## 대표값
1) 산술평균(mean)
- 계산이 쉽고 수학적으로 다루기 쉬움
- 모든 관측치를 사용하므로 특이값에 영향을 많이 받음
2) 중앙값(median)
- 관측한 자료를 순서대로 배열하여 가장 중앙에 있는 값
- 순위를 사용해 중앙에 있는 값만 사용하므로 특이값에 영향을 받지 않음
3) 최빈값(mode)
- 관측치 가운데 가장 여러 번 나타난 값
- 여러 개가 존재하거나 존재하지 않을 수 있고 중심을 잘 대변하지 못하는 경우가 많다
- 이산변수에 주로 사용하고 범주형 자료에도 사용 가능
- 분포가 한쪽으로 치우쳐있거나 특이값이 있는 경우 중앙값, 그렇지 않은 경우 산술평균이 적합하다

### 중앙값의 계산은 (관측치의 개수 + 1)/2 로 하여 해당 번째의 관측치의 값으로 한다 (짝수면 그냥 나누기 2 번째와 +1 번째의 평균으로 구한다)
- 치우치지 않은 그래프와 달리 오른쪽으로 치우친(오른쪽으로 긴 꼬리를 갖는) 그래프는 평균이 중앙값 보다 크다

## 산포도
1) 범위(range)
- 최대값-최소값
- 간단하지만 특이값에 큰 영향을 받음
2) 4분위 범위(IQR, interquartile range)
- 특이값에 영향 받지 않음
3) 표준편차
- 가장 널리 이용됨, 특이 값에 영향을 받음
- (변수값 - 평균값) 의 제곱 의 합계를 n-1 로 나누어 제곱근을 취함
4) 사분위 범위(IQR)
- 백분위수 : p백분위수란 p%의 관측치는 이 값 아래에 있고 나머지는 이 값보다 위에 있게되는 값
- 중앙값 : 50 백분위수
- Q1=25백분위수=제1사분위수
- Q3=75백분위수=제3사분위수
- 짝수 데이터 셋의 경우는 Q1,Q3 또한 해당 분위 수의 데이터와 +1 번째 데이터의 평균으로 구한다
- IQR : Q3 - Q1
- 다섯 숫자 요약(Five-number summary) : min Q1 median Q3 max
5) 상자그림(Boxplot)
- 다섯 숫자 요약의 graphical result
- 상자는 중앙 50%의 자료를 표시
- 여러 개의 분포를 한눈에 비교할 때 유용함
- Q1 과 Q3 로 끝나는 상자를 그린다 (상자의 길이 = IQR)
- 상자 안에 줄을 그어 중앙값을 표시한다
- Q3+1.5IQR 보다 크거나 Q1-1.5IQR 보다 작은 값은 * 또는 다른 심볼로 표시한다 : 1.5IQR criterion
- 상자의 끝에서 Outlier 가 아닌 값 중에 가장 큰 값과 가장 작은 값까지 줄을 긋는다
