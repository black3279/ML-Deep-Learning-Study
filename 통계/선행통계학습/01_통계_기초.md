# 러닝패킷: 통계의 기초
## Intro 1) 자료의 형태와 요약
1) (categorical) 범주형 자료 : 명목, 순서 자료
- 명목(Nominal) 변수 : 순서없는 범주를 가지는 변수
  - ex) 성별, 지역 등
- 순서(Ordinal) 변수 : 순서가 있는 범주를 가지는 변수
  - ex) 자동차 크기, 계층
2) (quantitative) 양적 자료 : 연속, 이산 자료
- 연속(Continuous) 변수 : 무수히 다른 값을 가짐
  - ex) 키, 몸무게, 온도
- 이산(Discrete) 변수 : 몇 개의 다른 값만 가짐
  - ex) 고장 횟수, 가족 구성원의 수

### 범주형 자료의 요약
- 도수분포표 : 혈액형 별 도수분포 등을 그린 분포표, 이름 보다는 자료의 수치를 기준으로 정렬하는 것이 직관적
- 막대그래프
- 파이 차트 등

### 양적 자료의 요약
- Graphical 요약 : 전체적인 분포 패턴과 벗어난 극단적 관측치를 살펴봄
- 수치적 요약
  - 대표값 : 산술평균, 중앙값 등
  - 산포도 : 범위, 표준편차 등
- Dotplot : 점포도로 나타냄
- 줄기-잎 그림(Stemplot) : 실제 자료 수치를 그대로 사용하면서 분포 형태를 보여줌
  - 정보 손실이 없고 데이터 양이 많지 않을 경우에 좋다
  - ex) 국가 - 성별 별 퍼센티지와 같은 그래프에서 십의 자리를 줄기로 일의 자리를 잎으로 표현할 수 있다
- 히스토그램 : 자료의 범위를 몇개의 구간으로 나누고 구간 별 관측치의 빈도 또는 상대 빈도 만을 나타내는 그림
  - 데이터 셋이 클 경우에 좋다
- Line graph(time plot) : 시계열 자료인 경우 x 축을 시간으로 한 time plot 에서 트렌드와 seasonal variation 등을 찾을 수 있다
