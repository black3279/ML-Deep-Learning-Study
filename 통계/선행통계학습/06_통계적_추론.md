# 러닝패킷: 통계의 기초
## 6) 통계적 추론
- 통계적 추론이란 표본의 정보를 이용하여 실제로 관찰되지 않은 부분을 포함한 모집단 전체에 대한 추측을 말한다
- 점추정이란 하나의 값으로 모수를 추정하는 것을 말한다
- 구간추정이란 모수가 속하리라고 생각되는 적절한 구간을 설정하여 추정하는 것이다

### 점추정
- 표본평균과 표본분산의 특징은 불편추정량이라는 것이다
- 불편추정량이란 표본평균의 평균은 모평균과 같고 표본분산의 평균은 모분산이라는 것이다

### 구간추정
- 중심극한정리에 의해 표본평균의 95% 정도가 모평균으로부터 2표준편차 이상 떨어져있지 않다
- 매우 많은 표본을 뽑아 계산했을때 95% 정도의 표본평균이 모평균에서 2 표준편차 이상 떨어져있지 않다는 것이다
- 정확히는 +-1.96 표준편차/제곱근n 만큼의 범위에서 95% 신뢰구간에 포함된다고 할 수 있다

### 신뢰수준 C인 신뢰구간
- 신뢰수준이 C인 신뢰구간 = 표본의 100*C% 에서 모평균을 포함하게 되는 신뢰구간
- 신뢰수준 C(100*C%) 인 신뢰구간 : x+- z*표본표준편차
- z* 값은 신뢰수준 C와 관계가 있으며 C 는 정규분포 곡선에서 -z* ~ z* 사이의 면적이다
- 신뢰구간의 길이 = 2 * z* 표본표준편차
- 즉, n = ( 2 * z* 모표준편차 / 신뢰구간의 길이) 제곱

### 모분산을 모를 때
- 표본표준편차 s로 모표준편차를 추정하여 사용한다
- 표본의 크기가 크면 표본은 모집단의 형태와 비슷해지고 작으면 달라질 수 있다

### t 분포
- 정규분포와 다르고 모표준편차를 모를 때 사용한다
- 표준편차가 알려져 있지 않으면 표본평균의 분포는 자유도가 n-1 인 t 분포를 따른다
- 즉 다음의 t 통계량의 분포는 자유도가 n-1 인 t 분포를 따른다
- t = (표본평균 - 모평균) / (표본표준편차 / n 제곱근) : 일표본 t 통계량
- n 이 크면 표본표준편차가 좋은 추정치가 되므로 t 분포는 정규분포와 매우 비슷한 분포가 된다

### 일표본 t-신뢰구간
- 신뢰수준 C(100*C%) 인 신뢰구간 : 표준평균 +- t(n-1)* 표본표준편차/n제곱근
- C는 t(n-1) 분포곡선에서 -t* ~ t* 사이의 면적이다
- C = 1-a 라 하면, t* 는 t(n-1) 분포에서 상위 a/2 critical value 이다
