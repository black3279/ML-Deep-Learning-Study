# 러닝패킷: 통계의 기초
## 4) 정규분포
- 정규분포는 연속형 분포 가운데 가장 많이 쓰이는 확률분포이다
- 정규분포는 N(평균, 분포) 로 많이 표현된다
- 표준편차가 작은 모집단의 분포는 평균 주위에 가깝게 몰려있고 큰 모집단의 분포는 넓게 퍼져있는 형태를 취한다
- 표준정규분포는 평균이 0 이고 표준편차가 1인 정규분포이다

### 68-95-99.7 규칙
- 평균에서 +- 표준편차 만큼 : 68%
- 평균에서 +- 2표준편차로 들어올 확률 : 95%
- 평균에서 +- 3표준편차로 들어올 확률 : 99.7%

### 정규분포의 표준화
- 모든 정규분포는 같은 형태적 성질을 갖기때문에 특정 확률분포를 표준화하여 확률계산을 할 수 있다
- 표준분포표를 사용하여 비율을 구할 수 있다
- z = (x - 평균) / 표준편차 로 하여 구할 수 있다
