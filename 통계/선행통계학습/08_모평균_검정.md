# 러닝패킷: 통계의 기초
## 8) 모평균에 대한 검정
### 모평균의 검정 : Z 검정
- 귀무가설은 모평균이 특정한 값이다
- 표본평균의 분포가 N(모평균, 표본표준편차) 를 따른다
- P-value 는 귀무가설이 맞다는 가정하에 표본으로부터 얻은 관측치 혹은 더 극단적인 관측치가 얻어질 확률이다
- z = (표본평균 - 모평균) / (모평균/n제곱근)

### 검정통계량
- 검정통계량은 귀무가설이 참일때 기대되는 값과 관측치 사이의 차이를 측정한다
- 관측치가 가정된 값으로부터 몇 표준편차만큼 떨어져있는가
- 양측검정 : 검정통계량이 아주 크거나 작을때 기각한다
- 단측검정 : 검정통계량이 아주크다/ 아주 작다 일때 기각한다
- 해당 값 혹은 해당 값보다 극단적인 값일 확률은 z 값을 구해 양측의 P-value 를 더해서 구한다
- 제 1종의 오류의 최대 허용 확률
- 유의수준이 결정되면 기각역이 결정된다
- 검정통계량의 값이 기각역에 속하면 p-value 가 a 보다 작아지므로 통계쩍 유의성을 보일 수 있다
- 모평균이 227g 인데 표본평균이 222g 이고 표준편차가 5g 이면 (n=4) 226.9 g 으로 모평균에 미치지 못하므로 95% 신뢰구간에서 기각한다

### 일표본 t-검정
- 모표준편차 를 모르는 경우에 사용한다
- 단맛의 감소 수준을 측정할 때 귀무가설은 단맛 감소의 모평균이 = 0 (변화가 없다) 이다
- 유의수준이 5% 라고 한다
- t = (표본평균 - 모평균) / (표본표준편차/n제곱근)
- 귀무가설 즉, 모평균이 0 이라고 하고 값을 구하면 2.70 이라고 하자
- 자유도는 9 이고 (n-1) 9 와 유의수준 5% 를 적용하면 1.833 으로 (단측검정 시) critical value 로 나오므로 해당 t 값이 1.833 보다 크므로 통계적으로 유의하여 귀무가설을 기각한다
- 양측 검정의 경우, 0.025 수준으로 critical value 를 계산한다
- 평균이 65 점보다 크다고 할 수 있는가 ? 는 귀무가설을 모평균은 65 이다 로 세운다
- 해당도 유의수준을 정해서 가설을 검증한다
- P-value 를 계산하여 해당 값이 유의수준보다 크다면 귀무가설을 기각할 수 없다 (작으면 기각한다)


### 실제적인 유의성
- 통계적 유의성은 랜덤표본에서 얻어진 결과에서 우연히 발생한지 아닌지만을 말해준다
- 통계적으로 유의하지만 효과가 있다는 것을 말해주고 그 크기를 말해주지 않기 때문에 특정 임계점을 기준으로 변화하는 크기의 값까지는 증명해주지 못한다
- 유의성이 발견되지 않았다는 것은 기각하지 않는 것이지 귀무가설을 옳다고 말하는 것은 아니다
- 충분한 정보가 없기 때문에 기각하지 않는 것이다
- 표본이 작을 경우에는 비모수적 방법과 같은 advanced skill 이 있다
